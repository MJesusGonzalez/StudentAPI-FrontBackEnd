# Student Admin web: backend (API) and frontend

## Packages

- node
- express
- firebase
- cors
- axios
- webpack

## How To Run

### Backend

1. Install packages
   ```
   npm init -y
   npm install
   ```
2. run backend
   ```
   node index.js
   ```

### Frontend
